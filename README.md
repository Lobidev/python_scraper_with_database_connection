## Setup

1. Repo clonen
   git clone ...
   cd mein_projekt

2. Virtuelles Environment erstellen
   python3 -m venv venv
   source venv/bin/activate

3. Pakete installieren
   pip install -r requirements.txt

4. .env einrichten
   cp .env.example .env
   # .env öffnen und Werte anpassen
